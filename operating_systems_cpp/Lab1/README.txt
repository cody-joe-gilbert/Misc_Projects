COMPILATION
The source code for the linker program is contained within the
C++ source file, "linker.cpp", and can be compiled using the
associated makefile, "makefile". The program was programmed and
tested with C++ 11, Version 4.4.7. The resulting executable 
will be "linker". 

EXECUTION
The "linker" program accepts one parameter "file", which is a 
string containing the pathname of the given input file
to be linked. 

OUTPUT
All output is sent to standard output to conform to the
grading shell.

EXCEPTIONS
The exceptions and warning given by the assignment are included,
as well as file I/O error messages for inappropriate input
files.
